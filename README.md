\# AlexGroup Catalog



Веб-приложение для каталога проектов домов.  

Проект реализован как fullstack-приложение на \*\*Next.js\*\*: клиентская часть отвечает за интерфейс каталога, а серверная часть — за API, работу с заявками и взаимодействие с базой данных.



\## О проекте



Система позволяет:



\- просматривать каталог объектов

\- открывать детальную страницу каждого объекта

\- фильтровать объекты по параметрам

\- сортировать каталог

\- сравнивать несколько объектов между собой

\- оставлять заявку по объекту

\- просматривать и обрабатывать заявки в панели заявок



Проект сделан в стиле современного коммерческого сайта с каталогом домов.



\## Основной функционал



\### Каталог объектов

На главной странице отображается список объектов с изображением, названием и основными характеристиками.



\### Фильтрация

Поддерживается фильтрация по следующим параметрам:



\- название

\- цена

\- площадь

\- спальни

\- комнаты

\- этажи



\### Сортировка

Есть сортировка по:



\- дате добавления

\- цене

\- площади



\### Пагинация

Каталог разбит на страницы, чтобы удобно работать с большим количеством объектов.



\### Страница объекта

Для каждого объекта доступна отдельная страница, где отображаются:



\- название

\- цена

\- площадь

\- этажность

\- количество спален

\- количество комнат

\- материал

\- описание

\- галерея изображений



\### Сравнение объектов

Пользователь может добавить несколько объектов в сравнение и увидеть их характеристики в общей таблице.



\### Заявки

На странице объекта можно оставить заявку через модальное окно.  

Заявка сохраняется в базу данных.



\### Работа с заявками

Есть отдельная страница `/leads`, где можно:



\- просматривать заявки

\- менять статус заявки

\- удалять заявку
